Detailed Description

template<typename T>
class StateSpaceController< T >

General State-Space Controller class.

See "example_StateSpaceController.cpp.example" to have an implementation example.

Assuming K is the controller: 

 K:

     | x_{i+1} = A*x_i + B*e_i
     |     u_i = C*x_i + D*e_i

         with e_i = r_i - y_i

QSMatrix class can be totally unused by user if a data file is generated to import the controller matrices. See loadControllerData() or StateSpaceController::StateSpaceController() for file format and more details.

This class can be used as a state space controller alone or coupled with a comparator to generate the error vector by using overloaded member functions.
